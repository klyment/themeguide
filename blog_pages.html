Blog pages

From your Shopify admin. Go to Online Store > Themes > Customize

Select to the blog page

Breadcrumb

Select the Breadcrumbs section

You will see a custom breadcrumbs panel

1. enable or disable breadcrumbs

2. If you want to use images for breadcrumbs. You can choose Use background image then enter an image for the breadcrumbs

3. You can add different breadcrumbs images to each collection
